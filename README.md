# 🚀 Veltrix Store

Um mini sistema frontend de afiliados focado em produtos da linha iPhone. O projeto simula uma loja moderna com interface elegante, navegação fluida e foco em conversão.

---

## 📱 Sobre o projeto

A **Veltrix Store** é uma aplicação frontend que apresenta produtos Apple (iPhone) como afiliado, com layout inspirado em e-commerces modernos.

A ideia é entregar uma experiência visual forte, responsiva e otimizada para destacar produtos premium.

---

## ✨ Funcionalidades

* 🛍️ Listagem de produtos (Fones, Relogios e Óculos)
* 🎯 Botões de afiliado (redirecionamento)
* 🌙 UI moderna com foco em conversão

---

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 🎨 Destaques

* Interface inspirada em grandes lojas digitais
* Componentes reutilizáveis
* Estilização moderna (gradientes, animações e microinterações)
* Experiência fluida tanto no desktop quanto no mobile
